-----------------------------------------------------------------------
-- Actor (entity)
-----------------------------------------------------------------------
Actor entity attributes:
- code
- name
- description
Actor relationships:
- use case
Use case relationship attributes:
- actor type
-----------------------------------------------------------------------
-- Code (attribute)
-----------------------------------------------------------------------
This attribute stores actor's code.
Constraints:
1) Actor code is unique in the use case model.
-----------------------------------------------------------------------
-- Name (attribute)
-----------------------------------------------------------------------
This attribute stores actor's name.
Constraints:
1) Actor name is unique in use case model.
-----------------------------------------------------------------------
-- Description (attribute)
-----------------------------------------------------------------------
This attribute stores actor's brief description.
Constraints:
1) Actor description is unique in use case model.
-----------------------------------------------------------------------
-- Use Case (relationship)
-----------------------------------------------------------------------
This relationship stores association actor with use case.
-----------------------------------------------------------------------
-- Actor Type (attribute)
-----------------------------------------------------------------------
This attribute stores actor type id.
Constraints:
1) Actor types:
a) Primary
b) Secondary
-----------------------------------------------------------------------
