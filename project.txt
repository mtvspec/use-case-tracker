Key Objects
- Project
- Project Team
- Project Team Member
--------------------------------------------------------------------------------
Team
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
Team Type
--------------------------------------------------------------------------------
- Группа заказчика (customer group)
- Группа разработчика (implementation group)
- Группа пользователей (users group)
- Группа сопровождения (maintain group)
- Группа внедрения (deployment group)
- Группа тестирования (test group)
- Группа обеспечения качества (software quality assurance group)
--------------------------------------------------------------------------------
Team Member
--------------------------------------------------------------------------------
Физическое лицо принятое в группу и имеющее определенную роль или роли в группе.
--------------------------------------------------------------------------------
Team Member Role
--------------------------------------------------------------------------------
- Руководитель группы
- Лидер продукта
- Постановщик
- Проектировщик
- Разработчик
- Архитектор
- Тестировщик
--------------------------------------------------------------------------------
Team Member Role Function
--------------------------------------------------------------------------------
Лидер продукта:
- принятие решений по продукту
Проектировщик:
- проектирование интерфейса
- концептуальное проектирование структуры данных
- логическое проектирование структуры данных
Разработчик:
- разработка интерфейса
- разработка бизнес-логики
- физическое проектирование структуры данных
Архитектор:
- разработка архитектуры системы
Тестировщик:
- разработка тестовых значений (test inputs)
- разработка ожидаемых результатов теста (expected result)
- оформление результатов тестирования (test results)
--------------------------------------------------------------------------------
