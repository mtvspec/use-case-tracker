--------------------------------------------------------------------------------
System (S)
--------------------------------------------------------------------------------
a) The combination of interacting elements organized to achieve one or more
stated purposes.
b) A set of interacting or interdependent component parts forming a
complex/intricate whole.
c) A set of rules that governs structure or behavior.
--------------------------------------------------------------------------------
Software system (SS)
--------------------------------------------------------------------------------
A system made up of software, hardware, and digital information, and that
provides its primary value by the execution of the system.
A software system can be part of a larger software, hardware, business or
social solution.
--------------------------------------------------------------------------------
Information System
--------------------------------------------------------------------------------
a) An organized system for the collection, organization, storage and
communication of information.
--------------------------------------------------------------------------------
Work System
--------------------------------------------------------------------------------
a) A system in which humans or (and) machines perform processes and
activities using resources to produce specific products or services for
customers.
--------------------------------------------------------------------------------
System Element
--------------------------------------------------------------------------------
A member of a set of elements that constitutes a system.
