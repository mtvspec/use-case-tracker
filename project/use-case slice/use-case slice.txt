-- Слайс
-==============================================================================-
Объект предназначен для моделирования слайсов - "кусочков" вариантов
использования (отдельных историй), которые проектная команда сначала
1) "очерчивает" - задает концепцию или границы "кусочка", затем
2) "подготавливает" - проектирует логику работы, интерфейсы и данные,
которые используются в этом "кусочке", затем
3) "анализирует" - изучает влияние "кусочка" на остальные компоненты системы, затем
4) "имплементирует" - разрабатывает "кусочек", затем
5) "верифицирует" - проверяет, тестирует "кусочек" и включает его в релиз.
На каждой из этих стадий проектная команда выполяет определенную работу,
в ходе выполнения которой она собирает нужную информацию, создает рабочие
продукты - артифакты, которые необходимо каждый в отдельности аккуратно
фиксировать в системе управления конфигурациями - хранить всю историю
создания "кусочка" системы и всей информацией, связанной с ним.
Данные стадии используется для оценки прогресса реализации проекта.
"Кусочки" на стадии "Верификация" являются объектами тестирования.
